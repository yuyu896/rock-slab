## Context

当前 `Organization.vue` 侧边栏使用 `buildOrgTree()` 构建树结构，根节点为"集团"，下设行政经理和区域节点。侧边栏工具栏只有"新增人员"按钮，左侧有空白。详情面板的操作按钮使用 `action-btn` 样式类，尺寸偏小。

## Goals / Non-Goals

**Goals:**
- 将"集团"从树结构移到工具栏作为标题，文字改为"启航事业部"
- 放大详情面板操作按钮尺寸

**Non-Goals:**
- 不改变树结构中区域、行政组、人员的层级关系
- 不修改后端 API

## Decisions

1. **标题放在工具栏左侧**
   - 理由：工具栏目前只有右侧一个按钮，左侧空白，放标题充分利用空间
   - 实现：工具栏改为 flex + space-between 布局，左侧放标题，右侧放按钮

2. **树构建逻辑调整**
   - 移除根节点的创建，`orgTree` 直接返回一个虚拟根或改为数组形式
   - 默认展开第一级（区域节点）改为在初始化时直接展开区域
   - 树渲染从 `orgTree.children` 开始遍历

3. **按钮尺寸调整**
   - 将 `action-btn` 的 padding 和图标尺寸增大

## Risks / Trade-offs

- **风险**: 移除根节点后，行政经理节点无处挂载 → **缓解**: 行政经理直接作为顶层节点显示

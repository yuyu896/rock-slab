## Why

组织架构模块存在 UI 冗余问题：1) 人员管理标签页与组织架构标签页的人员列表内容重复；2) 侧边栏顶部的"折叠架构/展开架构"按钮与树节点的展开收起功能重复。需要简化界面，提升用户体验。

## What Changes

- **BREAKING** 删除"人员管理"标签页（personnel tab）
- 在组织架构标签页的节点详情面板中添加"编辑"和"删除"两个操作按钮
- 删除侧边栏顶部的"折叠架构"和"展开架构"按钮
- 保留树节点自身的展开/收起交互（点击节点图标）

## Capabilities

### New Capabilities

- `org-node-actions`: 组织架构节点操作按钮（编辑、删除），位于节点详情面板

### Modified Capabilities

- `org-sidebar`: 移除折叠/展开全部按钮，简化侧边栏工具栏

## Impact

- **前端**: `Organization.vue` - 删除 personnel 标签页相关代码、删除折叠/展开按钮、在节点详情面板添加操作按钮
- **无后端变更**: 此次为纯前端 UI 简化

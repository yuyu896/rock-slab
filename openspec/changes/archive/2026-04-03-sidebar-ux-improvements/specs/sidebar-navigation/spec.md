# sidebar-navigation

## 概述

侧边栏导航组件，提供系统主要功能入口和用户信息展示。

## 功能需求

### F1. 用户信息区域

- **F1.1** 左下角展示用户头像、用户名、角色
- **F1.2** 点击用户信息区域，弹出下拉菜单
- **F1.3** 下拉菜单包含以下选项：
  - 修改头像：打开头像上传对话框
  - 修改个人信息：跳转到个人信息编辑页面
  - 退出登录：确认后退出系统并跳转到登录页

### F2. 导航菜单

- **F2.1** 菜单顺序（从上到下）：
  1. 工作台
  2. 统计报表
  3. 资产分类
  4. 资产管理（含子菜单：资产列表、采购入库、调拨记录）
  5. 资产盘点
  6. 组织架构
- **F2.2** 支持折叠/展开
- **F2.3** 当前页面菜单项高亮显示
- **F2.4** 支持徽章显示（如待审批数量）

### F3. 视觉要求

- **F3.1** 文字对比度符合 WCAG AA 标准（至少 4.5:1）
- **F3.2** 菜单文字清晰可读
- **F3.3** 支持明暗模式（如适用）

## 技术约束

- 使用 Vue 3 + TypeScript
- 使用 Element Plus 组件库（el-dropdown, el-dialog）
- 兼容移动端响应式布局

## 验收标准

1. 点击用户信息区域，下拉菜单正常弹出
2. 修改头像功能正常工作
3. 退出登录功能正常工作
4. 菜单顺序符合要求
5. 文字在标准显示器上清晰可读

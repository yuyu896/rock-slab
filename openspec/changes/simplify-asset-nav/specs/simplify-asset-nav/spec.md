## ADDED Requirements

### Requirement: Asset list as direct navigation item
The system SHALL display "资产列表" as a direct (non-collapsible) navigation item in the sidebar, linking to `/assets/list`.

#### Scenario: Sidebar shows asset list without expand
- **WHEN** user views the sidebar
- **THEN** "资产列表" is shown as a flat navigation item alongside other items, without an expand/collapse arrow

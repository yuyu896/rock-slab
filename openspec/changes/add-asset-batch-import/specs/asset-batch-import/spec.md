## ADDED Requirements

### Requirement: 批量导入按钮
资产列表页面操作栏 SHALL 在"导出"按钮旁显示"批量导入"按钮。按钮点击后 SHALL 打开批量导入弹窗。

#### Scenario: 点击批量导入按钮
- **WHEN** 用户在资产列表页面点击"批量导入"按钮
- **THEN** 系统打开批量导入弹窗

### Requirement: 下载导入模板
批量导入弹窗 SHALL 提供"下载模板"链接/按钮，点击后下载包含所有资产字段的 Excel 模板文件。

#### Scenario: 下载导入模板
- **WHEN** 用户在批量导入弹窗中点击"下载模板"
- **THEN** 系统下载一个 Excel 文件，包含资产字段的表头行（序号、分公司、分公司编号、资产编号、资产类目、物品分类、资产名称、规格、供应商、入库日期、是否租用、数量、单价、购入金额、出库日期、所属部门、使用人、当前状态、警戒线、是否充足、电脑序列号、备注）

### Requirement: 上传 Excel 文件
批量导入弹窗 SHALL 支持上传 .xlsx/.xls 格式的 Excel 文件。上传过程中 SHALL 显示 loading 状态。

#### Scenario: 上传 Excel 文件成功
- **WHEN** 用户选择 Excel 文件并确认上传
- **THEN** 系统调用后端导入接口，上传完成后显示导入结果

#### Scenario: 上传格式不正确的文件
- **WHEN** 用户选择了非 Excel 格式的文件
- **THEN** 系统提示"请上传 Excel 文件（.xlsx 或 .xls）"

### Requirement: 展示导入结果
导入完成后 SHALL 展示导入结果，包括成功导入数量和错误明细列表。

#### Scenario: 部分导入成功
- **WHEN** 导入完成且存在部分失败行
- **THEN** 系统显示"成功导入 N 条"，并列出每条失败的行号和错误原因

#### Scenario: 全部导入成功
- **WHEN** 导入完成且全部成功
- **THEN** 系统显示"成功导入 N 条"，自动关闭弹窗，刷新资产列表

#### Scenario: 全部导入失败
- **WHEN** 导入完成且全部失败
- **THEN** 系统显示"导入失败"，列出所有错误明细，用户可修正后重新上传

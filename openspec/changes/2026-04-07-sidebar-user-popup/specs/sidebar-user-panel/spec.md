# sidebar-user-panel

## 概述

侧边栏用户面板，点击左下角用户信息区域弹出，集成用户相关所有操作。

## 功能需求

### F1. 弹窗触发

- **F1.1** 点击侧边栏用户信息区域，弹出用户面板弹窗
- **F1.2** 点击弹窗外部区域或关闭按钮，关闭弹窗
- **F1.3** 弹窗显示时有遮罩层

### F2. 头像修改

- **F2.1** 头像区域可点击
- **F2.2** 点击后选择图片文件
- **F2.3** 选择后显示预览
- **F2.4** 确认后上传到服务器
- **F2.5** 上传成功后更新侧边栏头像显示

### F3. 个人信息编辑

- **F3.1** 显示当前用户姓名，可直接编辑
- **F3.2** 显示当前用户手机号（只读）
- **F3.3** 显示当前用户角色（只读）
- **F3.4** 提供保存按钮，保存修改

### F4. 修改密码

- **F4.1** 提供「修改密码」按钮
- **F4.2** 点击后展示密码修改表单：
  - 旧密码输入框
  - 新密码输入框
  - 确认密码输入框
- **F4.3** 表单验证：密码长度至少6位，两次输入一致
- **F4.4** 提交后调用 API 更新密码

### F5. 退出登录

- **F5.1** 提供「退出登录」按钮
- **F5.2** 点击后弹出确认对话框
- **F5.3** 确认后退出登录并跳转到登录页

## 技术约束

- 使用 Vue 3 + TypeScript
- 弹窗样式与设计系统保持一致
- 复用现有 API（uploadAvatar, updatePassword 等）

## 验收标准

1. 点击用户信息，弹窗正常弹出
2. 头像可正常更换
3. 姓名可修改并保存
4. 密码可正常修改
5. 退出登录功能正常
6. 弹窗样式美观，交互流畅

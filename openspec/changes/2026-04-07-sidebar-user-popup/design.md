# 侧边栏用户面板弹窗设计

## 交互流程

### 当前交互
```
点击用户区域 → 下拉菜单 → 点击"修改个人信息" → 跳转 /profile 页面
```

### 新交互
```
点击用户区域 → 弹出用户面板弹窗（包含所有功能）
```

## 弹窗设计

### 布局结构
```
┌─────────────────────────────────────┐
│  个人中心                        [X] │  ← 标题栏
├─────────────────────────────────────┤
│                                     │
│         [头像]                      │  ← 头像区域（可点击更换）
│          张三                       │
│       行政主管                       │
│                                     │
├─────────────────────────────────────┤
│  姓名        [张三          ]       │  ← 可编辑字段
│  手机号      138****1234            │  ← 只读（或可编辑）
├─────────────────────────────────────┤
│  [ 🔐 修改密码 ]                    │  ← 操作按钮
│  [ 🚪 退出登录 ]                    │
└─────────────────────────────────────┘
```

### 功能说明

#### 1. 头像修改
- 点击头像触发文件选择
- 支持预览
- 确认后上传

#### 2. 个人信息编辑
- 姓名：可直接编辑
- 手机号：显示（通常不可修改，需管理员操作）
- 其他基本信息

#### 3. 修改密码
- 点击后展开密码修改表单（或弹出子对话框）
- 包含：旧密码、新密码、确认密码

#### 4. 退出登录
- 点击后弹出确认对话框
- 确认后退出

## 样式规范

- 弹窗宽度：400px
- 圆角：12px
- 阴影：0 8px 32px rgba(0, 0, 0, 0.15)
- 背景：var(--color-bg-card)
- 文字：遵循设计系统变量

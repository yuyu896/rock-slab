# 统一流转模块操作按钮样式

## 问题

三个流转页面（采购入库、领用出库、调拨）的表格操作按钮样式不统一且不符合设计要求：

1. 当前按钮样式与页面不协调，大小、间距不合适
2. 三个页面需要完全一致的操作按钮样式
3. 按钮应该使用文字而非图标

## 设计规范

### 表格操作列按钮

三个页面（PurchaseList.vue、AssignList.vue、TransferList.vue）的操作列统一使用以下文字按钮：

| 按钮 | 默认样式 | 悬停样式 |
|------|----------|----------|
| 详情 | 灰色边框、灰色文字 | 绿色边框、绿色文字、浅绿背景 |
| 通过 | 绿色文字 + 绿色边框 | 绿色实底、白色文字 |
| 驳回 | 红色文字 + 红色边框 | 红色实底、白色文字 |

### 尺寸要求

- 按钮大小要与表格行高和页面整体风格协调
- 按钮之间间距合理（8px）
- 字体大小与表格内其他文字一致（12px）
- 按钮有合适的内边距（水平 10px，垂直 4px）

### 显示规则

- 「详情」按钮：始终显示
- 「通过」「驳回」按钮：仅当审批状态为"待审批"时显示

## 影响范围

- `frontend/src/views/transfers/PurchaseList.vue`
- `frontend/src/views/transfers/AssignList.vue`
- `frontend/src/views/transfers/TransferList.vue`
